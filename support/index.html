Support

先找到对应场景，再提交问题。

大多数兼容问题都与系统、Codex / ChatGPT 版本、安装来源或登录方式有关。选择对应指南通常能更快定位。

常用指南

多个 ChatGPT 账号

不同窗口如何保持独立登录状态。

 历史对话同步

项目、临时任务和多个来源如何选择。

 Windows 安装与排错

安装来源、权限、路径和启动问题。

 第三方 API 配置

Base URL、模型与 Responses API 兼容要求。

提交 GitHub Issue

反馈时请尽量包含：

  • Codex Multi Launcher 版本。
  • macOS 或 Windows 版本与芯片架构。
  • Codex / ChatGPT 桌面 App 版本和安装来源。
  • 使用 ChatGPT 账号还是 API Key。
  • 完整复现步骤、错误提示和必要截图。
  • App 内“复制诊断”生成的脱敏信息。
提交新 Issue搜索已有问题

公开反馈前检查隐私

请不要上传 API Key、ChatGPT Cookie、账号凭据、私人对话正文、公司内部项目代码或包含这些信息的完整配置文件。

诊断信息会主动排除 API Key，但截图和手工复制的文件仍需要自行确认。
