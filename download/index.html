安装与首次运行

  1. 先安装并至少启动一次官方 Codex / ChatGPT 桌面 App。
  2. 下载适合系统与芯片的 Codex Multi Launcher。
  3. 启动工具并在“设置”中确认检测到的桌面 App 路径。
  4. 创建第一个配置，选择账号登录或 API Key 模式。

如何确认下载可信

  • 下载地址域名应为 github.com/JqyModi/codex-multi-launcher。
  • Mac 可在系统信息中查看 Developer ID 签名与 Apple 公证状态。
  • 不要运行聊天群或网盘中来源不明的二次打包版本。
  • 不要在 Issue 中公开 API Key、Cookie 或私人历史对话。
