当前最新版

2026 年 7 月 17 日发布的 v0.1.6 重点解决 Codex / ChatGPT 新版状态持久化与官方账号登录问题。

  • 修复每次打开都重复显示迁移或初始化提示。
  • 修复重启后新项目和历史对话列表消失。
  • 新增 ChatGPT 官方账号登录配置，不再强制填写 API Key。
  • 账号登录配置隐藏接口地址和 API Key 编辑，避免误操作。
  • macOS 多架构包完成 Developer ID 签名与 Apple 公证。

应用内自动更新

App 冷启动时会检查新版本。检测到更新后会在首页提示，由用户决定是否查看、下载和安装；不会在没有确认的情况下强制更新。

Windows 当前未签名，升级时可能再次出现 SmartScreen 提示；macOS 更新包继续使用 Developer ID 签名与 Apple 公证。
