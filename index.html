Codex Multi Launcher
Codex 多开助手

同时运行多个相互隔离的 Codex / ChatGPT 桌面窗口。每个窗口可以登录不同账号、使用不同 API 配置，并按需带上已有项目和历史对话。

不只是“多开一个窗口”

把账号、配置和工作记录一起隔离。

手动复制目录或反复修改配置，容易造成登录状态串用、项目消失或历史对话无法继续。Codex 多开助手为每个配置建立独立数据目录，并在创建时明确选择账号方式、接口配置和需要同步的历史范围。

常见问题

下载前需要知道的几件事。

它会修改官方 Codex / ChatGPT App 吗？

不会替换官方 App。工具通过独立的配置目录和应用数据目录启动对应窗口。

ChatGPT 账号模式还需要填写 API Key 吗？

不需要。创建后在独立窗口内完成账号登录即可，接口地址和 API Key 输入也会隐藏。

历史对话同步后，来源配置会被修改吗？

不会。同步会为新配置建立独立副本，后续新增对话不会反向影响来源。

所有第三方 API 都能使用吗？

不能保证。服务商需要实际支持 Codex 使用的 Responses API；只支持 Chat Completions 的接口可能无法使用。
