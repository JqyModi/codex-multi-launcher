ChatGPT Plus / Pro · Codex 账号登录

不填写 API Key，也能创建独立 Codex 配置。

选择“ChatGPT 账号登录”后，工具只准备独立应用环境。账号登录在官方 Codex / ChatGPT 窗口内完成。

创建步骤

  1. 创建新 Profile 并填写名称。
  2. 选择“ChatGPT 账号登录”。
  3. 根据需要选择历史对话同步来源和范围。
  4. 生成并打开 Profile。
  5. 在出现的官方窗口中登录对应账号。
账号模式不写入 API Key，不生成 auth.json API Key 配置，也不会导出 OPENAI_API_KEY。

切换账号

需要更换某个 Profile 的账号时，只在对应窗口中退出并重新登录。不要在 Profile 详情页填写接口地址或 API Key；账号模式下这些输入不会显示。

如果两个 Profile 登录了同一个账号，它们仍使用独立本地应用数据，但云端账号内容和额度可能来自同一个账号。

账号登录和历史对话是两件事

登录账号决定当前窗口使用哪个 ChatGPT / OpenAI 身份；历史同步决定创建时复制哪些本地项目和对话记录。可以只登录新账号而不同步历史，也可以在创建时同时完成两项操作。

账号安全

  • 只在官方 Codex / ChatGPT 窗口中输入账号信息。
  • 多开助手不会要求你提供账号密码或 Cookie。
  • 不要把 Profile 的完整应用数据目录分享给他人。
  • 删除 Profile 前确认其中是否有仍需保留的本地工作记录。
